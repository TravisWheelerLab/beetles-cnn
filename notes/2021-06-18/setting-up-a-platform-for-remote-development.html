PyCharm

We downloaded pycharm professional with a student license.

Then, we followed the steps on https://stackoverflow.com/questions/32211813/pycharm-remote-development to get pycharm connected to bioinfo and gscc.

Kayla's password for bioinfo: beetles Kayla's password for gscc: netid password

running files on bioinfo

  • open up a mac terminal, and ssh into bioinfo
  • cd into the beetles/ directory
  • make sure the relevant files are uploaded with PyCharm
  • If you get errors, make absolute paths relative, and make sure they're the right ones for the remote machine
  • make sure the (base) environment is activated on bioinfo
  • run your code with ''python <my-code-here>''
