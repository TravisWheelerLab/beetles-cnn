Beetlegeuse is a collection of tools aimed at using deep learning to classify beetle chirps in .wav files.

Quickstart:
```
pip install beetles
```
Learn how to use the tools provided in this package [here](https://github.com/TravisWheelerLab/beetles-cnn/wiki).
